Продвинутый редактор скинов оружия и перчаток для Unreal Engine. 
Виджет позволяет просматривать, настраивать и создавать скины с полным контролем над материалами, паттернами, износом, цветами и визуальными эффектами

  • Просмотр и управление большой библиотекой скинов

  • Поддержка CS2- и CS:GO-моделей

  • Полноценная настройка материалов в реальном времени

  • Поддержка стилей: Anodized, Spraypaint, Case Hardened и других

  • Режимы Glitter, Pearlescence, Overlay, Spraypaint Halftone и другие

  • Анимированные UV для паттернов с готовыми пресетами

  • Самообновляемая система добавления новых скинов

  • Unreal Engine 5.6 или новее
  • PyToolkit plugin (версии 1.8 и выше)

  • Установленная Counter-Strike 2

  • Утилита Source2Viewer CLI для декомпиляции моделей и текстур

  • Свободное место на диске (первый запуск может импортировать несколько тысяч текстур)

Скачайте и распакуйте плагин PyToolkit (минимум 1.8 версии) в папку Plugins в корневой директории Unreal Engine (C:\Program Files\Epic Games\UE_5.6\Engine\Plugins).

Запустите Unreal Engine и включите следующие плагины (в меню Edit > Plugins):

  • Editor Scripting Utilities
  • Python Editor Script Plugin
  • Python Foundation Packages
  • Http Blueprint
  • PyToolkit

Затем перезапустите движок.

  1. Скачайте архив с виджетом
  2. Распакуйте архив в любую папку — это уже готовый проект. Если нужно добавить виджет в существующий проект, откройте в скачанном проекте папку Content и скопируйте все её файлы в папку Content вашего проекта.
  3. Откройте проект.
  4. В Content Browser найдите папку Widgets. Кликните по виджету S2 Skins Manager правой кнопкой мыши и выберите Run Editor Utility Widget.
Виджет готов к работе.
При первом запуске будет предложено заполнить поля с настройками и импортировать все найденные скины.

  • Decompiler Executable
    Путь к декомпилеру Source2Viewer CLI (обычно называется cli-windows-x64.zip).

  • Game Folder
    Путь к установленной CS2 (например: D:/SteamLibrary/steamapps/common/Counter-Strike Global Offensive).

  • Decompiled Folder
    Любая папка, куда будут складываться  декомпилированные файлы из игры (главное, чтобы на диске хватало места).

После импорта всех скинов можно будет выбрать любой скин для любого оружия, а так же воспользоваться полем поиска

Если возникают ошибки — убедитесь, что:

  • Виджет установлен по правильному пути в проекте;
  • Включены все необходимые плагины;
  • Используется Unreal Engine версии 5.6 или выше (в имени архива с виджетом указана минимально нужная версия движка);
  • Декомпилер обновлён до последней версии.
    Если декомпилер не работает, можно попробовать использовать DEV версию.

После сохранения (или запекания — для старых перчаток) материал скина появляется в Content Browser.
Чтобы применить материал к оружию или перчаткам, нужно:

  • перетащить материал прямо на модель в окне просмотра (Viewport),
  • либо назначить его вручную в панели Details, заменив соответствующий слот материала, как показано на скриншоте выше.
  1. Кнопка возврата (Back) — возвращает пользователя к списку скинов.
  2. Флаг “Texture View” — показывает UV развертку.
  3. Падающее меню —  переключатель режима отображения (Lit, Roughness, Metallic и др.).
  4. Флаг “CS2 Model” — включает отображение модели оружия версии CS2.
  5. Окно предпросмотра — реалтайм-отображение материала и всех изменений. Поддержка вращения, зума и смены угла обзора

  8. Настройка UV паттернов и потертостей
  9. FX-эффекты — настройки Glitter, Case Hardened, Pearlescence, Overlay, Halftone и Emissive.

Main Settings
Paint Style — определяет базовый стиль рендеринга скина. От выбранного стиля зависит набор доступных параметров, логика наложения паттернов и поведение материала.

Pearlescence — включает перламутровый эффект. Цвет поверхности меняется в зависимости от угла обзора и освещения. Используется для скинов с выраженным «переливающимся» покрытием.

Glitter — активирует эффект блесток в материале. Добавляет микроскопические отражающие элементы, создающие эффект искрящейся поверхности.

Use Overlay — включает дополнительный слой поверх основного материала. Используется для графических элементов или эффектов, не влияя на базовый слой скина.

Case Hardening — активирует режим имитации закалки металла. Формирует характерный многоцветный узор с плавными цветовыми переходами.

Use Emissive — включает самосветящийся слой материала. Позволяет добавить элементы, светящиеся независимо от освещения сцены.

Wear

Wear — основной параметр износа покрытия. Определяет степень стирания краски и проявления нижнего слоя материала.

Wear Multiplier — множитель износа. Усиливает или ослабляет влияние параметра Wear без изменения его базового значения.

Wear Softness — контролирует плавность переходов между изношенными и неизношенными участками. Низкие значения дают резкие края, высокие — более мягкий износ.

Paint Durability — маска прочности покрытия. Определяет, какие области поверхности изнашиваются быстрее, а какие дольше сохраняют краску.

Texture Adjustment

Basecolor Top Layer Brightness — управляет яркостью верхнего цветового слоя (скина).

Basecolor Top Layer Contrast — регулирует контраст верхнего цветового слоя, делая рисунок более резким или более мягким.

Underlayer Brightness — задает яркость нижнего слоя материала (оружия).

Underlayer Contrast — управляет контрастом нижнего слоя материала.

Metal Color Brightness — управляет яркостью на металлических областях скина.

Color

Color 0 (Base Coat) — базовый слой краски, который покрывает все окрашиваемые поверхности оружия. Является основой для всех последующих цветовых слоёв и всегда применяется первым.

Color 1 (Red Mask) — цвет, накладываемый поверх базового слоя с использованием красного канала паттерна в качестве маски.

Color 2 (Green Mask) — цвет, накладываемый поверх слоя Red Mask с использованием зелёного канала паттерна в качестве маски.

Color 3 (Blue Mask) — цвет, накладываемый поверх слоя Green Mask с использованием синего канала паттерна в качестве маски. Является финальным цветовым слоем в последовательности наложения.

Case Hardening Color — работает только со стилем Case Hardening. Определяет, какой из цветовых слоёв должен быть заменен эффектом Case Hardening.

Color Exponent — усиливает глубину и насыщенность цветов без прямого изменения яркости.

Color Hue Shift — сдвигает оттенок всей цветовой гаммы по цветовому кругу.

Color Saturation — управляет насыщенностью цветов. Низкие значения делают цвет более приглушенным, высокие - более ярким.

Use AO In Final Color — учитывает текстуру Ambient Occlusion при формировании финального цвета, добавляя глубину и визуальный объем.

Roughness

Roughness From Pattern — использует значения шероховатости непосредственно из паттерна. Позволяет управлять микрорельефом поверхности на основе текстуры паттерна.

Roughness Uses Pattern UVs — применяет UV-координаты паттерна для текстуры шероховатости. 

Use Roughness by Color — включает управление шероховатостью на основе цветовой маски. Разные цветовые области паттерна могут иметь различную степень шероховатости.

Roughness Toplayer Min — минимальное значение шероховатости верхнего слоя покрытия (скина).

Roughness Toplayer Max — максимальное значение шероховатости верхнего слоя покрытия (скина).

Roughness Underlayer Min — минимальное значение шероховатости нижнего слоя материала (оружия), проявляющегося при износе.

Roughness Underlayer Max — максимальное значение шероховатости нижнего слоя материала (оружия).

Roughness Mask Power — степень влияния маски шероховатости. Чем выше значение, тем сильнее контраст между гладкими и матовыми участками.

Use Roughness Texture — использует отдельную текстуру шероховатости вместо процедурных значений.

Roughness by Color — включает управление шероховатостью для каждого цветового слоя, сформированного на основе масок из текстуры и Paint-By-Number масок. Позволяет задавать разную шероховатость для отдельных цветовых областей скина.

Paint Roughness — базовое значение шероховатости краски. Используется, если не заданы дополнительные источники шероховатости. Зависит от маски скина.

Metalness

Metalness Toplayer Min — минимальное значение металличности верхнего слоя покрытия (скина).

Metalness Toplayer Max — максимальное значение металличности верхнего слоя покрытия.

Metalness Underlayer Min — минимальное значение металличности нижнего слоя материала (оружия).

Metalness Underlayer Max — максимальное значение металличности нижнего слоя материала.

Metalness Mask Power — степень влияния маски металличности. Определяет контраст между металлическими и неметаллическими участками.

Paint Metalness — базовое значение металличности краски. Используется как основа для формирования металлических свойств покрытия. Зависит от маски скина.

Pattern UV

Pattern Scale — задает масштаб паттерна на поверхности оружия. Меньшие значения увеличивают размер элементов узора, большие — делают рисунок более мелким.

Pattern Rotation — поворачивает паттерн вокруг центра, изменяя ориентацию рисунка на поверхности модели.

Pattern Offset U — смещает паттерн по горизонтальной UV-координате.

Pattern Offset V — смещает паттерн по вертикальной UV-координате.

Animated UVs — включает анимацию UV-координат паттерна. Используется для создания динамических эффектов и движущихся узоров. В дальнейшем можно будет выбрать один из 9 анимированных пресетов.

Pattern Rotation Center U — задаёт точку центра вращения паттерна по оси U.

Pattern Rotation Center V — задаёт точку центра вращения паттерна по оси V.

Pattern Scope Scale — дополнительный параметр масштабирования паттерна. Используется только для оружия SSG08.

Mesh Bounds — масштаб модели оружия. Используется только в triplanar стилях (3 и 6).

Wear UV

 Те же самые настройки, что и для паттерна, но для текстуры износа на поверхности скина.

Grunge UV

 Те же самые настройки, что и для паттерна, но для текстуры загрязнений и шумов на поверхности скина.

Glitter

Glitter Intensity — управляет интенсивностью эффекта блёсток. Определяет, насколько заметны отражающие частицы на поверхности материала.

Glitter Scale — задаёт масштаб узора блёсток. Большие значения делают частицы более мелкими, меньшие — более крупными.

Glitter Rainbow Balance — регулирует баланс цветного перелива блёсток, смещая эффект в сторону радужных оттенков.

Glitter Rainbow Spread — управляет разбросом радужных цветов в эффекте блёсток, влияя на степень цветового разнообразия.

Case Hardening

Case Hardening Geometric Influence — определяет степень влияния текстуры Culvature на формирование узора закалки металла.

Case Hardening Pattern Influence — задаёт влияние паттерна на распределение эффекта закалки по поверхности.

Case Hardening Ramp Offset — смещает градиент цветового перехода закалки по оси Y, изменяя цвета переливов.

Case Hardening Trilinear — включает режим triplanar для эффекта закалки.

Pearlescence

Pearlescence Mask Uses Pattern UVs — использует UV-координаты паттерна для маски перламутрового эффекта.

Pearlescent on Metallic Only — ограничивает эффект перламутрового перелива только металлическими участками материала.

Use Pearlescence Mask — включает использование пользовательской маски для управления областью действия перламутрового эффекта.

Pearlescent Scale — определяет степень смещения оттенка цвета поверхности в зависимости от угла между поверхностью и направлением камеры.

Pearlescence Color Intensity — управляет интенсивностью цветного перелива перламутрового эффекта.

Pearlescence Shift — смещает оттенок перламутрового перелива, изменяя его цветовой характер.

Overlay

Overlay Uses Pattern UVs — использует UV-координаты паттерна для overlay текстуры.

Overlay Blend Mode — определяет режим смешивания overlay-слоя с базовым материалом. Доступны режимы: прозрачное наложение, замена оттенка и насыщенности, умножение цветов, сложение цветов и наложение слоя поверх albedo.

Overlay Mask Mode — определяет, какая маска используется для ограничения области применения overlay-слоя.
Доступные режимы: 0  (без маски), 1  (базовый слой), 2 (красный канал маски), 3 (зелёный канал маски), 4 (синий канал маски), 5 (комбинация всех RGB каналов), 6 (маска паттерна).

Overlay Brightness — управляет яркостью overlay-слоя.

Overlay Durability — определяет степень устойчивости overlay-слоя к wear-износу.

Overlay Material Strength — задаёт силу влияния overlay-материала на финальный результат.

Overlay Metalness — управляет металличностью overlay-слоя.

Overlay Roughness — задаёт шероховатость overlay-слоя.

Overlay Pearlescent Mask — определяет влияние перламутрового эффекта на overlay-слой.

Overlay Albedo Strength — определяет силу и заметность overlay-текстуры в финальном результате.

Halftone

Spraypaint Halftone — включает эффект полутонов (альфа канал паттерна), имитирующий spraypaint-покрытие.

Halftone in Cavity — ограничивает эффект полутонов текстурой Cavity.

Halftone Cavity Cutoff — задаёт пороговое значение Cavity текстуры, при котором плотность полутонов достигает максимума.

Halftone Rotation Offset X — смещает вращение полутонов для красного канала маски.

Halftone Rotation Offset Y — смещает вращение полутонов для зелёного канала маски.

Halftone Rotation Offset Z — смещает вращение полутонов для синего канала маски.

Halftone Thresholds — влияет на мягкость или резкость границ полутонов.

Halftone Pattern Levels — Влияет на плотность полутонов.

Emissive

Emissive Use Mask Texture — использует текстуру маски для ограничения области подсветки скина. Позволяет применять эффект только к выбранным участкам оружия.

Emissive Intensity — задаёт интенсивность подсветки.

Emissive Mask Power — управляет контрастом и силой влияния маски подсветки.

Emissive Use Pattern — включает режим управления подсветкой через цветовые каналы паттерна. Позволяет задавать разные области свечения на основе масок Base, Red, Green и Blue.

Emissive Black — определяет интенсивность свечения для базового (чёрного) слоя паттерна.

Emissive R — определяет интенсивность свечения для области, заданной красным каналом паттерна.

Emissive G — определяет интенсивность свечения для области, заданной зелёным каналом паттерна.

Emissive B — определяет интенсивность свечения для области, заданной синим каналом паттерна.

Emissive Mask Texture — текстура маски, определяющая форму и распределение свечения.

Emissive Use Texture — включает использование отдельной текстуры свечения вместо процедурных или паттерн-масок.

Emissive Texture — текстура, используемая для формирования рисунка свечения.

Emissive Color — цвет свечения.

Emissive Use Flickering — включает эффект мерцания свечения.

Emissive Flickering Frequency — задаёт частоту мерцания свечения.

Emissive Animated UVs — включает анимацию UV-координат для текстуры подсветки.

Emissive Speed X — скорость анимации UV-координат по оси X.

Emissive Speed Y — скорость анимации UV-координат по оси Y.

Textures

Override Ambient Occlusion — позволяет переопределить текстуру Ambient Occlusion пользовательской текстурой вместо стандартной.

Override Masks — включает использование пользовательской текстуры масок, заменяя стандартные маски оружия.

Override Metalness — позволяет переопределить текстуру металличности отдельной текстурой.

Override Roughness — позволяет переопределить текстуру шероховатости отдельной текстурой.

Override Position — включает использование пользовательской текстуры Position Map вместо стандартной.

Override Base Color — позволяет заменить базовую текстуру albedo пользовательской текстурой.

Use All Masks — принудительно использует все доступные маски независимо от выбранного стиля и активных эффектов (только для стиля triplanar).

Use Custom Normal Map — включает использование пользовательской текстуры нормалей.

Use SFX Mask — включает использование SFX-маски для управления эффектами, такими как перламутровость, поверхностная закалка и блёстки.

Pattern Settings

Use Pattern — использование паттерна для формирования цвета и деталей поверхности.

Use Pattern = 0 — Pattern Disabled — паттерн полностью отключён. Use Pattern = 1 — Pattern Replace (Selective Replacement) — паттерн используется как замена одного конкретного цвета палитры. — цвет, указанный в Pattern Replace Index, исключается из палитры и заменяется паттерном. — остальные цвета палитры остаются нетронутыми. Use Pattern = 2 — Raw Pattern (Unfiltered Pattern) — паттерн используется как есть, без привязки к палитре.
— Pattern Color Indices не используются. Use Pattern = 3 — Pattern Overlay (Full Palette Modulation) — паттерн применяется ко всей палитре целиком.
— каждый цвет палитры смешивается с паттерном, а не заменяется выборочно.

Pattern Color Indices — массив индексов, определяющий, какие цвета палитры используются для каждого цветового канала паттерна.

Pattern Detail Influence — управляет тем, насколько паттерн влияет на микродетализацию поверхности и смешивание слоёв.

Pattern Paint Thickness — задаёт виртуальную толщину слоя краски паттерна. Влияет на затенение, нормали и взаимодействие с износом.

Flip Fixup — коррекция направления нормалей.

Pattern Replace Index — указывает, какой именно цвет палитры должен быть заменён паттерном в режимах, где паттерн участвует в замене.

Sample Slice — определяет какие слои текстур из массива используются при выборке grunge, detail и detail normal.

Pattern

Surface Color Saturation 1–4 — Насыщенность основной текстуры слоя N. 0 — чёрно-белая. Surface Color Tint 1–4 — Цвет и сила окраски основной поверхности слоя N. Surface Grime Min Max 1–4 — Диапазон износа для накопления грязи на основной поверхности слоя N. R = начало появления грязи, G = максимальная загрязнённость.
Adv - Substrate
Substrate Color Brightness 1–4 — Яркость нижнего слоя материала (подложки) слоя N. Substrate Color Contrast 1–4 — Контрастность подложки слоя N. Substrate Color Saturation 1–4 — Насыщенность подложки слоя N. Substrate Composite Color Translucency 1–4 — Насколько сильно зональный цвет окрашивает подложку слоя N. 0 — подложка не окрашивается, 1 — полная окраска. Substrate Color Tint 1–4 — Цветовой оттенок подложки слоя N. Substrate Burnishing Min Max 1–4 — Диапазон износа для эффекта полировки на подложке слоя N. R = начало эффекта, G = максимальная интенсивность. Substrate Grime Min Max 1–4 — Диапазон износа для накопления грязи на подложке слоя N. R = начало появления грязи, G = максимальная загрязнённость.
Pattern Texture — Рисунок/декаль на перчатке. RGB = цвет, A = маска прозрачности (компрессия: BC7, sRGB вкл). Damage Texture 1–4 — Маска царапин и потёртостей слоя N. Только R-канал — где светлее, там больше повреждений (компрессия: Grayscale, sRGB выкл). Grime Texture 1–4 — Текстура грязи и пыли слоя N. RGB = цвет грязи, накладываемой на изношенные участки (компрессия: BC7, sRGB вкл). Substrate Texture 1–4 — Нижний слой материала (кожа, пластик, ткань), просвечивающий сквозь царапины и износ. RGB = цвет, A = маска окраски зонами (компрессия: BC7, sRGB вкл). Substrate Normal 1–4 — Карта нормалей нижнего слоя. Задаёт рельеф подложки в изношенных зонах (компрессия: BC7, sRGB выкл). Substrate Properties 1–4 — PBR-данные подложки. R = min шероховатость, G = металличность, B = cloth-фактор, A = нижний порог диапазона повреждений (компрессия: BC7, sRGB выкл). Surface Texture 1–4 — Основная текстура поверхности слоя N. RGB = цвет, A = маска окраски зонами (компрессия: BC7, sRGB вкл). Surface Normal 1–4 — Карта нормалей основной поверхности слоя N. X/Z каналы также хранят диапазон шероховатости (компрессия: BC7, sRGB выкл). Surface Properties 1–4 — PBR-данные поверхности. R = min шероховатость, G = металличность, B = cloth-фактор, A = верхний порог повреждений (компрессия: BC7, sRGB выкл). Layer ID — Маска слоёв. R/G/B = веса слоёв 1/2/3, вес слоя 4 вычисляется автоматически. Определяет, какой материал виден в каждой точке (компрессия: BC7, sRGB выкл). Normal Map — Базовая карта нормалей объекта. Задаёт рельеф меша — швы, складки, тиснение (компрессия: BC7, sRGB выкл). Object Properties — Запечённые данные объекта. R = AO (тени в углах), G = запечённые повреждения, B = сила износа в пикселе (компрессия: BC7, sRGB выкл). Pattern Properties — PBR-свойства паттерна. R = металличность, G = шероховатость слоя краски (компрессия: BC7, sRGB выкл). Tint ID Texture — Карта зон окраски. R-канал кодирует одну из 8 зон, каждую из которых можно покрасить независимо (компрессия: Grayscale, sRGB выкл).
