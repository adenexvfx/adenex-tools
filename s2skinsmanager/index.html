S2 Skins Manager
Widget Виджет v3.21
Advanced CS2 weapon skin and glove editor for Unreal Engine. Browse, configure, and create skins with full control over materials, patterns, wear, colors, and visual effects.
Продвинутый редактор скинов оружия и перчаток для Unreal Engine. Просматривайте, настраивайте и создавайте скины с полным контролем над материалами, паттернами, износом, цветами и визуальными эффектами.

  1. Download and extract the PyToolkit plugin (v1.8+) into the Plugins folder of your Unreal Engine installation: C:\Program Files\Epic Games\UE_5.6\Engine\Plugins.
  2. Launch Unreal Engine and enable the following plugins (Edit > Plugins):
    • Editor Scripting Utilities
    • Python Editor Script Plugin
    • Python Foundation Packages
    • Http Blueprint
    • PyToolkit
    Restart the engine.
  3. Download the widget archive — it's a ready-to-use project. To add the widget to an existing project, copy the Content folder from the archive into your project's Content folder.
  4. In the Content Browser, find the Widgets folder, right-click S2 Skins Manager, and select Run Editor Utility Widget.
On first launch, configure the paths and import all skins:
  • Decompiler Executable — Path to the Source2Viewer CLI decompiler.
  • Game Folder — Path to your CS2 installation.
  • Decompiled Folder — Folder for decompiled game files.

Main Settings
  • Paint Style — Base rendering style. Controls available parameters and material behavior.
  • Pearlescence — Pearlescent color shift based on viewing angle.
  • Glitter — Microscopic reflective glitter effect.
  • Use Overlay — Additional layer on top of the base material.
  • Case Hardening — Metal hardening simulation with multicolor patterns.
  • Use Emissive — Self-illuminated material layer.
Wear
  • Wear — Paint erosion level.
  • Wear Multiplier — Multiplier for the Wear parameter.
  • Wear Softness — Transition smoothness between worn and unworn areas.
  • Paint Durability — Mask determining which areas wear faster.
Color
  • Color 0 (Base Coat) — Base paint layer.
  • Color 1 (Red Mask) — Color over the base, masked by the pattern's red channel.
  • Color 2 (Green Mask) — Color masked by the green channel.
  • Color 3 (Blue Mask) — Final color layer, masked by the blue channel.
  • Case Hardening Color — Which color layer is replaced by Case Hardening effect.
  • Color Hue Shift — Shifts overall hue around the color wheel.
  • Color Saturation — Controls color saturation.
Main Settings

Wear
  • Wear — Степень стирания краски.
  • Wear Multiplier — Множитель износа.
  • Wear Softness — Плавность переходов между изношенными и неизношенными участками.
  • Paint Durability — Маска прочности покрытия.
Color
  • Color 0 (Base Coat) — Базовый слой краски.
  • Color 1 (Red Mask) — Цвет поверх базового, маска — красный канал паттерна.
  • Color 2 (Green Mask) — Цвет, маска — зелёный канал паттерна.
  • Color 3 (Blue Mask) — Финальный цветовой слой, маска — синий канал.

  • Color Saturation — Насыщенность цветов.
Pattern UV
  • Pattern Scale — Pattern scale on the weapon surface.
  • Pattern Rotation — Rotates the pattern.
  • Pattern Offset U / V — Shifts the pattern along UV axes.
  • Animated UVs — Animated UV coordinates. Choose from 9 presets.
Glitter

If errors occur, make sure:
  • The widget is installed at the correct path in your project.
  • All required plugins are enabled.
  • Unreal Engine 5.6 or newer is used.
  • The decompiler is updated to the latest version. If it doesn't work, try the DEV version.
Если возникают ошибки — убедитесь, что:
  • Виджет установлен по правильному пути в проекте.
  • Включены все необходимые плагины.
  • Используется Unreal Engine версии 5.6 или выше.
  • Декомпилер обновлён до последней версии. Если не работает — попробуйте DEV версию.
