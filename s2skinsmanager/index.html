Main Settings
Paint Style — определяет базовый стиль рендеринга скина. От выбранного стиля зависит набор доступных параметров, логика наложения паттернов и поведение материала.

Pearlescence — включает перламутровый эффект. Цвет поверхности меняется в зависимости от угла обзора и освещения. Используется для скинов с выраженным «переливающимся» покрытием.

Glitter — активирует эффект блесток в материале. Добавляет микроскопические отражающие элементы, создающие эффект искрящейся поверхности.

Use Overlay — включает дополнительный слой поверх основного материала. Используется для графических элементов или эффектов, не влияя на базовый слой скина.

Case Hardening — активирует режим имитации закалки металла. Формирует характерный многоцветный узор с плавными цветовыми переходами.

Use Emissive — включает самосветящийся слой материала. Позволяет добавить элементы, светящиеся независимо от освещения сцены.

Wear

Wear — основной параметр износа покрытия. Определяет степень стирания краски и проявления нижнего слоя материала.

Wear Multiplier — множитель износа. Усиливает или ослабляет влияние параметра Wear без изменения его базового значения.

Wear Softness — контролирует плавность переходов между изношенными и неизношенными участками. Низкие значения дают резкие края, высокие — более мягкий износ.

Paint Durability — маска прочности покрытия. Определяет, какие области поверхности изнашиваются быстрее, а какие дольше сохраняют краску.

Texture Adjustment

Basecolor Top Layer Brightness — управляет яркостью верхнего цветового слоя (скина).

Basecolor Top Layer Contrast — регулирует контраст верхнего цветового слоя, делая рисунок более резким или более мягким.

Underlayer Brightness — задает яркость нижнего слоя материала (оружия).

Underlayer Contrast — управляет контрастом нижнего слоя материала.

Metal Color Brightness — управляет яркостью на металлических областях скина.

Color

Color 0 (Base Coat) — базовый слой краски, который покрывает все окрашиваемые поверхности оружия. Является основой для всех последующих цветовых слоёв и всегда применяется первым.

Color 1 (Red Mask) — цвет, накладываемый поверх базового слоя с использованием красного канала паттерна в качестве маски.

Color 2 (Green Mask) — цвет, накладываемый поверх слоя Red Mask с использованием зелёного канала паттерна в качестве маски.

Color 3 (Blue Mask) — цвет, накладываемый поверх слоя Green Mask с использованием синего канала паттерна в качестве маски. Является финальным цветовым слоем в последовательности наложения.

Case Hardening Color — работает только со стилем Case Hardening. Определяет, какой из цветовых слоёв должен быть заменен эффектом Case Hardening.

Color Exponent — усиливает глубину и насыщенность цветов без прямого изменения яркости.

Color Hue Shift — сдвигает оттенок всей цветовой гаммы по цветовому кругу.

Color Saturation — управляет насыщенностью цветов. Низкие значения делают цвет более приглушенным, высокие - более ярким.

Use AO In Final Color — учитывает текстуру Ambient Occlusion при формировании финального цвета, добавляя глубину и визуальный объем.

Main Settings
Paint Style — defines the base rendering style of the skin. The selected style determines the available parameters, pattern blending logic, and overall material behavior.

Pearlescence — enables a pearlescent effect. The surface color shifts depending on the viewing angle and lighting conditions. Used for skins with a pronounced iridescent finish.

Glitter — enables a glitter effect in the material. Adds microscopic reflective particles, creating a sparkling surface appearance.

Use Overlay — enables an additional layer on top of the base material. Used for graphical elements or effects without affecting the underlying skin layer.

Case Hardening — activates a metal hardening simulation mode. Generates a characteristic multicolor pattern with smooth color transitions.

Use Emissive — enables a self-illuminated material layer. Allows adding elements that glow independently of scene lighting.

Wear

Wear — the primary wear parameter. Controls the level of paint erosion and exposure of the underlying material layer.

Wear Multiplier — a wear intensity multiplier. Amplifies or reduces the effect of the Wear parameter without changing its base value.

Wear Softness — controls the smoothness of transitions between worn and unworn areas. Lower values produce sharper edges; higher values create softer wear.

Paint Durability — a durability mask for the coating. Defines which areas wear out faster and which retain paint longer.

Texture Adjustment

Basecolor Top Layer Brightness — controls the brightness of the top color layer (the skin).

Basecolor Top Layer Contrast — adjusts the contrast of the top color layer, making the pattern sharper or softer.

Underlayer Brightness — sets the brightness of the underlying material layer (the weapon).

Underlayer Contrast — controls the contrast of the underlying material.

Metal Color Brightness — controls brightness in metallic areas of the skin.

Color

Color 0 (Base Coat) — the base paint layer applied to all paintable surfaces of the weapon. Serves as the foundation for all subsequent color layers and is always applied first.

Color 1 (Red Mask) — a color applied over the base layer using the red channel of the pattern as a mask.

Color 2 (Green Mask) — a color applied over the Red Mask layer using the green channel of the pattern as a mask.

Color 3 (Blue Mask) — a color applied over the Green Mask layer using the blue channel of the pattern as a mask. This is the final color layer in the blending sequence.

Case Hardening Color — used only with the Case Hardening style. Determines which color layer is replaced by the Case Hardening effect.

Color Exponent — enhances color depth and intensity without directly changing brightness.

Color Hue Shift — shifts the hue of the entire color palette along the color wheel.

Color Saturation — controls color intensity. Lower values produce more muted tones; higher values create more vibrant colors.

Use AO In Final Color — incorporates the Ambient Occlusion texture into the final color output, adding depth and visual volume.

Roughness

Roughness From Pattern — использует значения шероховатости непосредственно из паттерна. Позволяет управлять микрорельефом поверхности на основе текстуры паттерна.

Roughness Uses Pattern UVs — применяет UV-координаты паттерна для текстуры шероховатости. 

Use Roughness by Color — включает управление шероховатостью на основе цветовой маски. Разные цветовые области паттерна могут иметь различную степень шероховатости.

Roughness Toplayer Min — минимальное значение шероховатости верхнего слоя покрытия (скина).

Roughness Toplayer Max — максимальное значение шероховатости верхнего слоя покрытия (скина).

Roughness Underlayer Min — минимальное значение шероховатости нижнего слоя материала (оружия), проявляющегося при износе.

Roughness Underlayer Max — максимальное значение шероховатости нижнего слоя материала (оружия).

Roughness Mask Power — степень влияния маски шероховатости. Чем выше значение, тем сильнее контраст между гладкими и матовыми участками.

Use Roughness Texture — использует отдельную текстуру шероховатости вместо процедурных значений.

Roughness by Color — включает управление шероховатостью для каждого цветового слоя, сформированного на основе масок из текстуры и Paint-By-Number масок. Позволяет задавать разную шероховатость для отдельных цветовых областей скина.

Paint Roughness — базовое значение шероховатости краски. Используется, если не заданы дополнительные источники шероховатости. Зависит от маски скина.

Metalness

Metalness Toplayer Min — минимальное значение металличности верхнего слоя покрытия (скина).

Metalness Toplayer Max — максимальное значение металличности верхнего слоя покрытия.

Metalness Underlayer Min — минимальное значение металличности нижнего слоя материала (оружия).

Metalness Underlayer Max — максимальное значение металличности нижнего слоя материала.

Metalness Mask Power — степень влияния маски металличности. Определяет контраст между металлическими и неметаллическими участками.

Paint Metalness — базовое значение металличности краски. Используется как основа для формирования металлических свойств покрытия. Зависит от маски скина.

Roughness

Roughness From Pattern — uses roughness values directly from the pattern. Allows control over surface microdetail based on the pattern texture.

Roughness Uses Pattern UVs — applies the pattern's UV coordinates to the roughness texture.

Use Roughness by Color — enables roughness control based on a color mask. Different color regions of the pattern can have varying roughness levels.

Roughness Toplayer Min — minimum roughness value for the top coating layer (skin).

Roughness Toplayer Max — maximum roughness value for the top coating layer (skin).

Roughness Underlayer Min — minimum roughness value for the underlying material layer (weapon), revealed through wear.

Roughness Underlayer Max — maximum roughness value for the underlying material layer (weapon).

Roughness Mask Power — controls the influence of the roughness mask. Higher values increase contrast between smooth and matte areas.

Use Roughness Texture — uses a separate roughness texture instead of procedural values.

Roughness by Color — enables roughness control per color layer, generated from texture masks and Paint-By-Number masks. Allows different roughness values for individual color regions of the skin.

Paint Roughness — base roughness value of the paint. Used when no additional roughness sources are specified. Depends on the skin mask.

Metalness

Metalness Toplayer Min — minimum metalness value for the top coating layer (skin).

Metalness Toplayer Max — maximum metalness value for the top coating layer.

Metalness Underlayer Min — minimum metalness value for the underlying material layer (weapon).

Metalness Underlayer Max — maximum metalness value for the underlying material layer.

Metalness Mask Power — controls the influence of the metalness mask. Defines contrast between metallic and non-metallic areas.

Paint Metalness — base metalness value of the paint. Serves as the foundation for defining the coating's metallic properties. Depends on the skin mask.

Pattern UV

Pattern Scale — задает масштаб паттерна на поверхности оружия. Меньшие значения увеличивают размер элементов узора, большие — делают рисунок более мелким.

Pattern Rotation — поворачивает паттерн вокруг центра, изменяя ориентацию рисунка на поверхности модели.

Pattern Offset U — смещает паттерн по горизонтальной UV-координате.

Pattern Offset V — смещает паттерн по вертикальной UV-координате.

Animated UVs — включает анимацию UV-координат паттерна. Используется для создания динамических эффектов и движущихся узоров. В дальнейшем можно будет выбрать один из 9 анимированных пресетов.

Pattern Rotation Center U — задаёт точку центра вращения паттерна по оси U.

Pattern Rotation Center V — задаёт точку центра вращения паттерна по оси V.

Pattern Scope Scale — дополнительный параметр масштабирования паттерна. Используется только для оружия SSG08.

Mesh Bounds — масштаб модели оружия. Используется только в triplanar стилях (3 и 6).

Wear UV

 Те же самые настройки, что и для паттерна, но для текстуры износа на поверхности скина.

Grunge UV

 Те же самые настройки, что и для паттерна, но для текстуры загрязнений и шумов на поверхности скина.

Pattern UV

Pattern Scale — sets the scale of the pattern on the weapon surface. Lower values increase the size of pattern elements, while higher values make the pattern finer.

Pattern Rotation — rotates the pattern around its center, changing the orientation of the design on the model surface.

Pattern Offset U — shifts the pattern along the horizontal UV coordinate.

Pattern Offset V — shifts the pattern along the vertical UV coordinate.

Animated UVs — enables animation of the pattern's UV coordinates. Used to create dynamic effects and moving patterns. You can choose from 9 animated presets.

Pattern Rotation Center U — defines the rotation center of the pattern along the U axis.

Pattern Rotation Center V — defines the rotation center of the pattern along the V axis.

Pattern Scope Scale — an additional pattern scaling parameter. Used only for the SSG08 weapon.

Mesh Bounds — controls the scale of the weapon model. Used only in triplanar styles (3 and 6).

Wear UV

Same settings as for the pattern, but applied to the wear texture on the skin surface.

Grunge UV

Same settings as for the pattern, but applied to the dirt and noise texture on the skin surface.

Glitter

Glitter Intensity — управляет интенсивностью эффекта блёсток. Определяет, насколько заметны отражающие частицы на поверхности материала.

Glitter Scale — задаёт масштаб узора блёсток. Большие значения делают частицы более мелкими, меньшие — более крупными.

Glitter Rainbow Balance — регулирует баланс цветного перелива блёсток, смещая эффект в сторону радужных оттенков.

Glitter Rainbow Spread — управляет разбросом радужных цветов в эффекте блёсток, влияя на степень цветового разнообразия.

Case Hardening

Case Hardening Geometric Influence — определяет степень влияния текстуры Culvature на формирование узора закалки металла.

Case Hardening Pattern Influence — задаёт влияние паттерна на распределение эффекта закалки по поверхности.

Case Hardening Ramp Offset — смещает градиент цветового перехода закалки по оси Y, изменяя цвета переливов.

Case Hardening Trilinear — включает режим triplanar для эффекта закалки.

Pearlescence

Pearlescence Mask Uses Pattern UVs — использует UV-координаты паттерна для маски перламутрового эффекта.

Pearlescent on Metallic Only — ограничивает эффект перламутрового перелива только металлическими участками материала.

Use Pearlescence Mask — включает использование пользовательской маски для управления областью действия перламутрового эффекта.

Pearlescent Scale — определяет степень смещения оттенка цвета поверхности в зависимости от угла между поверхностью и направлением камеры.

Pearlescence Color Intensity — управляет интенсивностью цветного перелива перламутрового эффекта.

Pearlescence Shift — смещает оттенок перламутрового перелива, изменяя его цветовой характер.

Overlay

RGB = цвет грязи, накладываемой на изношенные участки (компрессия: BC7, sRGB вкл). Substrate Texture 1–4 — Нижний слой материала (кожа, пластик, ткань), просвечивающий сквозь царапины и износ. RGB = цвет, A = маска окраски зонами (компрессия: BC7, sRGB вкл). Substrate Normal 1–4 — Карта нормалей нижнего слоя. Задаёт рельеф подложки в изношенных зонах (компрессия: BC7, sRGB выкл). Substrate Properties 1–4 — PBR-данные подложки. R = min шероховатость, G = металличность, B = cloth-фактор, A = нижний порог диапазона повреждений (компрессия: BC7, sRGB выкл). Surface Texture 1–4 — Основная текстура поверхности слоя N. RGB = цвет, A = маска окраски зонами (компрессия: BC7, sRGB вкл). Surface Normal 1–4 — Карта нормалей основной поверхности слоя N. X/Z каналы также хранят диапазон шероховатости (компрессия: BC7, sRGB выкл). Surface Properties 1–4 — PBR-данные поверхности. R = min шероховатость, G = металличность, B = cloth-фактор, A = верхний порог повреждений (компрессия: BC7, sRGB выкл). Layer ID — Маска слоёв. R/G/B = веса слоёв 1/2/3, вес слоя 4 вычисляется автоматически. Определяет, какой материал виден в каждой точке (компрессия: BC7, sRGB выкл). Normal Map — Базовая карта нормалей объекта. Задаёт рельеф меша — швы, складки, тиснение (компрессия: BC7, sRGB выкл). Object Properties — Запечённые данные объекта. R = AO (тени в углах), G = запечённые повреждения, B = сила износа в пикселе (компрессия: BC7, sRGB выкл). Pattern Properties — PBR-свойства паттерна. R = металличность, G = шероховатость слоя краски (компрессия: BC7, sRGB выкл). Tint ID Texture — Карта зон окраски. R-канал кодирует одну из 8 зон, каждую из которых можно покрасить независимо (компрессия: Grayscale, sRGB выкл).
Pattern Texture — the pattern/decal on the glove. RGB = color, A = transparency mask (compression: BC7, sRGB on). Damage Texture 1–4 — scratch and wear mask for layer N. R channel only — lighter areas = more damage (compression: Grayscale, sRGB off). Grime Texture 1–4 — dirt and dust texture for layer N. RGB = dirt color applied to worn areas (compression: BC7, sRGB on). Substrate Texture 1–4 — the bottom material layer (leather, plastic, fabric) showing through scratches and wear. RGB = color, A = zone tinting mask (compression: BC7, sRGB on). Substrate Normal 1–4 — normal map of the bottom layer. Defines substrate relief in worn zones (compression: BC7, sRGB off). Substrate Properties 1–4 — PBR data for the substrate. R = min roughness, G = metalness, B = cloth factor, A = lower threshold of the damage range (compression: BC7, sRGB off). Surface Texture 1–4 — main surface texture of layer N. RGB = color, A = zone tinting mask (compression: BC7, sRGB on). Surface Normal 1–4 — normal map of the main surface of layer N. X/Z channels also store the roughness range (compression: BC7, sRGB off). Surface Properties 1–4 — PBR data for the surface. R = min roughness, G = metalness, B = cloth factor, A = upper damage threshold (compression: BC7, sRGB off). Layer ID — layer mask. R/G/B = weights of layers 1/2/3, weight of layer 4 is calculated automatically. Defines which material is visible at each point (compression: BC7, sRGB off). Normal Map — base normal map of the object. Defines mesh relief — seams, folds, embossing (compression: BC7, sRGB off). Object Properties — baked object data. R = AO (corner shadows), G = baked damage, B = wear strength per pixel (compression: BC7, sRGB off). Pattern Properties — PBR properties of the pattern. R = metalness, G = roughness of the paint layer (compression: BC7, sRGB off). Tint ID Texture — tint zone map. R channel encodes one of 8 zones, each of which can be colored independently (compression: Grayscale, sRGB off).
