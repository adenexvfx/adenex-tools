Adenex / AFXCameraPath
AFXCameraPath
Unreal Engine Plugin Unreal Engine Plugin v1.4
Editor plugin for building camera paths in Unreal Engine. Create, select, and manipulate keyframes directly in the viewport with stable GUID-based selection and click-based interaction via hit proxies.
Редакторный плагин для построения путей камеры в Unreal Engine. Создавайте, выбирайте и управляйте кейфреймами прямо во вьюпорте — с устойчивой идентификацией по GUID и кликовым взаимодействием через hit proxies.

  1. Download and extract the plugin archive.
  2. Place the AFXCameraPath folder into the Plugins folder of your UE5 project (create it if it doesn't exist): YourProject/Plugins/AFXCameraPath.
  3. Open the project in Unreal Engine. The plugin will compile automatically on first launch.
  4. Enable the plugin in Edit > Plugins, then restart the engine.
  5. The AFXCameraPath panel will appear in the editor toolbar.

  1. Open the AFXCameraPath panel from the toolbar.
  2. Click Add Key to add a keyframe at the current camera position.
  3. Click on any keyframe in the viewport to select it. Use the standard UE transform gizmo to reposition.
  4. Adjust FOV and other per-key parameters in the panel.
  5. The camera path spline is drawn in the viewport in real time.
