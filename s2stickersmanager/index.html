S2 Stickers Manager
ВиджетWidget v2.1
Виджет для просмотра, редактирования и импорта стикеров и граффити из CS2 для Unreal Engine. Полный контроль над материалами, износом, цветами и визуальными эффектами. A widget for viewing, editing, and importing stickers and graffiti from Counter-Strike 2 into Unreal Engine. The widget allows you to configure and create assets with full control over materials, wear, colors, and visual effects.

Скачайте и распакуйте плагин PyToolkit (минимум 1.8 версии) в папку Plugins в корневой директории Unreal Engine (C:\Program Files\Epic Games\UE_5.6\Engine\Plugins). Запустите Unreal Engine и включите следующие плагины (в меню Edit > Plugins):
  • Editor Scripting Utilities
  • Python Editor Script Plugin
  • Python Foundation Packages
  • Http Blueprint
  • PyToolkit
Затем перезапустите движок.
  1. Скачайте архив с виджетом
  2. Распакуйте содержимое архива. Откройте в скачанном архиве папку Content и скопируйте все её файлы в папку Content вашего проекта.
  3. В Content Browser найдите папку Widgets. Кликните по виджету S2_StickersManager правой кнопкой мыши и выберите Run Editor Utility Widget.
Виджет готов к работе.
При первом запуске будет предложено заполнить поля с настройками и импортировать все найденные скины.

  • Decompiler Executable
    Путь к декомпилеру Source2Viewer CLI (обычно называется cli-windows-x64.zip).

  • Game Folder
    Путь к установленной CS2 (например: D:/SteamLibrary/steamapps/common/Counter-Strike Global Offensive).

  • Decompiled Folder
    Любая папка, куда будут складываться  декомпилированные файлы из игры (главное, чтобы на диске хватало места).

После импорта всех стикеров можно будет выбрать любой стикер из нужной капсулы, а также воспользоваться полем поиска

Download and extract the PyToolkit plugin (version 1.8 or higher) into the Plugins folder in the root directory of Unreal Engine (C:\Program Files\Epic Games\UE_5.6\Engine\Plugins).

Launch Unreal Engine and enable the following plugins (via Edit > Plugins):

  • Editor Scripting Utilities
  • Python Editor Script Plugin
  • HTTP Blueprint
  • PyToolkit

Then restart the engine.

  1. Download the widget archive
  2. Extract the contents of the archive. Open the Content folder in the downloaded archive and copy all its files into your project's Content folder.
  3. In the Content Browser, locate the Widgets folder. Right-click the S2_StickersManager widget and select Run Editor Utility Widget.

The widget is ready to use.

On the first launch, you'll be prompted to fill in the settings fields and import all detected stickers and graffiti.

  • Decompiler Executable
    Path to the Source2Viewer CLI decompiler (usually named cli-windows-x64.zip).

  • Game Folder
    Path to your installed CS2 (for example: D:/SteamLibrary/steamapps/common/Counter-Strike Global Offensive).

  • Decompiled Folder
    Any folder where decompiled game files will be stored (make sure you have enough disk space).

After all stickers are imported, you'll be able to select any sticker from a specific capsule and use the search field to find what you need.

  • Включены все необходимые плагины;
  • Используется Unreal Engine версии 5.6 или выше (в имени архива с виджетом указана минимально нужная версия движка);
  • Декомпилер обновлён до последней версии.
    Если декомпилер не работает, можно попробовать использовать DEV версию.

If you encounter errors, make sure that:

  • The widget is installed in the correct location within the project;
  • All required plugins are enabled;
  • You are using Unreal Engine version 5.6 or newer (the minimum required version is specified in the widget archive name);

    If the decompiler does not work, try using the DEV version.

    Позволяет включить или скрыть фон, чтобы оценить форму, прозрачность и края стикера.
  3. Окно предпросмотра — реалтайм-отображение материала и всех изменений. Поддержка вращения, зума и смены угла обзора
  4. Основные настройки материала
  5. Параметры отображения материала — настройка Roughness / Metalness / Визуальных эффектов
  6. Текстуры.
  7. Панель управления камерой.
  8. Save — сохранение финального Material Instance для дальнейшего использования.
  1. Back button — returns the user to the previous screen (sticker list or category).
  2. "Show Backing" toggle — controls the visibility of the sticker's backing layer.
    Allows you to show or hide the background to better evaluate the shape, transparency, and edges of the sticker.
  3. Preview window — real-time display of the material and all changes. Supports rotation, zoom, and viewing angle adjustments.
  4. Core material settings
  5. Material display parameters — adjust Roughness, Metalness, and visual effects.
  6. Textures
  7. Camera controls panel
  8. Save — saves the final Material Instance for later use.
Mode

Определяет активные визуальные режимы материала стикера:

  • Glitter — добавляет эффект блёсток и микроскопических отражений.

  • Holographic — включает голографический эффект с переливом цвета в зависимости от угла обзора.

  • Metallic — включает металлический тип отображения.

  • Paper Backing — включает бумажную подложку, имитирующую классический стикер.

Wear

Параметры износа стикера:

  • Wear — общий уровень износа. Чем выше значение, тем сильнее эффект старения.

  • Wear Bias — смещает распределение износа по поверхности.

  • Wear Scratches — управляет интенсивностью царапин и микроповреждений.

Color

Настройки цветовой обработки:

  • Automatic PBR Color Fitting — автоматически приводит цвета к корректному PBR-диапазону.

  • Legacy Tint Multiply — включает старый режим умножения цвета (для совместимости).

  • Color Tint — основной цвет стикера.

  • Tint Saturate — насыщенность цветового оттенка.

  • Color Boost — усиливает общий цветовой контраст.

  • SFX Color Boost — усиливает цвет эффектов (голография, блёстки).

  • Hue Shift — смещает оттенок по цветовому кругу.

Mode

Defines the active visual modes of the sticker material:

  • Glitter — adds a sparkle effect with microscopic reflections.

  • Holographic — enables a holographic effect with color shifting based on the viewing angle.

  • Metallic — enables a metallic rendering mode.

  • Paper Backing — enables a paper backing that simulates a classic sticker.

Wear

Sticker wear parameters:

  • Wear — overall wear level. Higher values result in stronger aging effects.

  • Wear Bias — shifts the distribution of wear across the surface.

  • Wear Scratches — controls the intensity of scratches and micro-damage.

Color

Color processing settings:

  • Automatic PBR Color Fitting — automatically adjusts colors to a valid PBR range.

  • Legacy Tint Multiply — enables the legacy color multiply mode (for compatibility).

  • Color Tint — the base color of the sticker.

  • Tint Saturate — saturation level of the color tint.

  • Color Boost — increases overall color contrast.

  • SFX Color Boost — enhances effect colors (holographic, glitter).

  • Hue Shift — shifts the hue along the color wheel.

Roughness

Настройки шероховатости верхнего слоя:

  • Roughness Min — минимальная шероховатость поверхности (более гладкий и отражающий вид).

  • Roughness Max — максимальная шероховатость поверхности (более матовый вид).

  • Preserve Roughness — сохраняет исходное значение шероховатости из текстуры.

Metalness

Настройки характера отражений материала:

  • Metalness Min — минимальный уровень металлического поведения поверхности.

  • Metalness Max — максимальный уровень металлического поведения поверхности.

SFX

Параметры визуальных эффектов:

  • Normal Strength — сила нормалей.

  • Glitter Scale — масштаб эффекта блёсток (меньше значение — крупнее элементы, больше — мельче).

  • Glitter Normal Strength — сила нормалей у эффекта блёсток.

  • Use Emissive — включает подсветку элементов стикера.

  • Clamp Spectrum V — ограничивает диапазон цветового спектра для голографического эффекта.

  • Holo Exponent — управляет резкостью и контрастом голографического эффекта.

  • Holo X/Y Offset — смещение голографического эффекта.

Roughness

Top layer roughness settings:

  • Roughness Min — minimum surface roughness (smoother, more reflective appearance).

  • Roughness Max — maximum surface roughness (more matte appearance).

  • Preserve Roughness — preserves the original roughness value from the texture.

Metalness

Material reflection characteristics:

  • Metalness Min — minimum level of metallic behavior for the surface.

  • Metalness Max — maximum level of metallic behavior for the surface.

SFX

Visual effects parameters:

  • Normal Strength — normal map intensity.

  • Glitter Scale — scale of the glitter effect (lower values produce larger particles; higher values produce finer detail).

  • Glitter Normal Strength — normal map intensity for the glitter effect.

  • Use Emissive — enables emissive lighting for sticker elements.

  • Clamp Spectrum V — limits the color spectrum range for the holographic effect.

  • Holo Exponent — controls sharpness and contrast of the holographic effect.

  • Holo X/Y Offset — offsets the holographic effect.

Textures

Sticker Texture — основная цветная текстура стикера. Определяет форму, цвет и базовый рисунок.

Normal Roughness Masks — комбинированная текстура, содержащая данные для нормалей и шероховатости поверхности.

Glitter Normal Texture — текстура нормалей для эффекта блёсток. Определяет, как отдельные блёстки отражают свет.

Holo Spectrum Texture — текстура для голографического эффекта.

SFX Mask Texture — маска голографического эффекта.

Sticker Scratches — текстура царапин и повреждений. Используется для добавления реалистичного износа и неровностей поверхности.

Textures

Sticker Texture — the main color texture of the sticker. Defines its shape, color, and base design.

Normal Roughness Masks — a combined texture containing normal and surface roughness data.

Glitter Normal Texture — a normal map for the glitter effect. Defines how individual particles reflect light.

Holo Spectrum Texture — a texture used for the holographic effect.

SFX Mask Texture — a mask texture for the holographic effect.

Sticker Scratches — a texture for scratches and damage. Used to add realistic wear and surface imperfections.
