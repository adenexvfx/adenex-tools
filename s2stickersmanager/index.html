Adenex / S2 Stickers Manager
S2 Stickers Manager
Widget Виджет v2.1
Widget for viewing, editing, and importing CS2 stickers and graffiti into Unreal Engine. Full control over materials, wear, colors, and visual effects including Glitter, Holographic, Metallic, and Paper styles.
Виджет для просмотра, редактирования и импорта стикеров и граффити из CS2 для Unreal Engine. Полный контроль над материалами, износом, цветами и визуальными эффектами.

  1. Download and extract the PyToolkit plugin (v1.8+) into the Plugins folder of your Unreal Engine installation: C:\Program Files\Epic Games\UE_5.6\Engine\Plugins.
  2. Launch Unreal Engine and enable the following plugins (Edit > Plugins):
    • Editor Scripting Utilities
    • Python Editor Script Plugin
    • Python Foundation Packages
    • Http Blueprint
    • PyToolkit
    Restart the engine.
  3. Download the widget archive. Open the archive, navigate to the Content folder, and copy all its files into the Content folder of your project.
  4. In the Content Browser, find the Widgets folder, right-click S2_StickersManager, and select Run Editor Utility Widget.
On first launch, configure the paths and import all stickers:
  • Decompiler Executable — Path to the Source2Viewer CLI decompiler.
  • Game Folder — Path to your CS2 installation (e.g. D:/SteamLibrary/steamapps/common/Counter-Strike Global Offensive).
  • Decompiled Folder — Any folder for decompiled game files. Make sure you have enough free disk space.

UI Overview
  1. Back button — Returns to the sticker list or category.
  2. Show Backing — Toggles the sticker's paper backing to preview shape, transparency, and edges.
  3. Preview window — Real-time material display with rotation, zoom, and view controls.
  4. Main material settings
  5. Material display settings — Roughness / Metalness / Visual effects.
  6. Textures
  7. Camera panel
  8. Save — Saves the final Material Instance.
Main Tab — Mode
  • Glitter — Adds glitter and micro-reflection effects.
  • Holographic — Color shift based on viewing angle.
  • Metallic — Metallic display type.
  • Paper Backing — Paper backing imitating a classic sticker.
Main Tab — Wear
  • Wear — Overall wear level.
  • Wear Bias — Shifts wear distribution across the surface.
  • Wear Scratches — Intensity of scratches and micro-damage.
Main Tab — Color
  • Automatic PBR Color Fitting — Brings colors into correct PBR range.
  • Color Tint — Primary sticker color.
  • Tint Saturate — Tint saturation.
  • Color Boost — Increases color contrast.
  • SFX Color Boost — Boosts effect colors (holographic, glitter).
  • Hue Shift — Shifts hue around the color wheel.
Shading Tab
  • Roughness Min / Max — Surface roughness range.
  • Preserve Roughness — Preserves original roughness from the texture.
  • Metalness Min / Max — Metallic behavior range.
  • Normal Strength — Normal map strength.
  • Glitter Scale — Glitter effect scale.
  • Use Emissive — Enables sticker element glow.
  • Holo Exponent — Sharpness and contrast of the holographic effect.
  • Holo X/Y Offset — Holographic effect position offset.
Textures Tab
  • Sticker Texture — Main color texture.
  • Normal Roughness Masks — Combined normal and roughness texture.
  • Glitter Normal Texture — Normal texture for glitter effect.
  • Holo Spectrum Texture — Texture for holographic effect.
  • SFX Mask Texture — Holographic effect mask.
  • Sticker Scratches — Scratch and damage texture.
Общая структура интерфейса
  1. Кнопка возврата (Back) — Возвращает к списку стикеров или категории.
  2. Show Backing — Переключатель отображения задней подложки стикера.
  3. Окно предпросмотра — Реалтайм-отображение материала с ротацией, зумом и сменой угла.
  4. Основные настройки материала
  5. Параметры отображения материала — Roughness / Metalness / Визуальные эффекты.
  6. Текстуры
  7. Панель управления камерой
  8. Save — Сохранение финального Material Instance.
Вкладка Main — Mode
  • Glitter — Добавляет эффект блёсток и микроскопических отражений.

Вкладка Main — Wear

  • Wear Bias — Смещение распределения износа по поверхности.
  • Wear Scratches — Интенсивность царапин и микроповреждений.
Вкладка Main — Color
  • Automatic PBR Color Fitting — Автоматически приводит цвета к корректному PBR-диапазону.
  • Color Tint — Основной цвет стикера.
  • Tint Saturate — Насыщенность цветового оттенка.
  • Color Boost — Усиливает общий цветовой контраст.
  • SFX Color Boost — Усиливает цвет эффектов (голография, блёстки).
  • Hue Shift — Смещает оттенок по цветовому кругу.
Вкладка Shading
  • Roughness Min / Max — Диапазон шероховатости поверхности.
  • Preserve Roughness — Сохраняет исходную шероховатость из текстуры.
  • Metalness Min / Max — Диапазон металличности.
  • Normal Strength — Сила нормалей.
  • Glitter Scale — Масштаб эффекта блёсток.
  • Use Emissive — Включает подсветку элементов стикера.
  • Holo Exponent — Резкость и контраст голографического эффекта.
  • Holo X/Y Offset — Смещение голографического эффекта.
Вкладка Textures
  • Sticker Texture — Основная цветная текстура стикера.
  • Normal Roughness Masks — Комбинированная текстура нормалей и шероховатости.
  • Glitter Normal Texture — Текстура нормалей для эффекта блёсток.
  • Holo Spectrum Texture — Текстура для голографического эффекта.
  • SFX Mask Texture — Маска голографического эффекта.

If errors occur, make sure:
  • The widget is installed at the correct path in your project.
  • All required plugins are enabled.
  • Unreal Engine 5.6 or newer is used.
  • The decompiler is updated to the latest version. If it doesn't work, try the DEV version.
Если возникают ошибки — убедитесь, что:
  • Виджет установлен по правильному пути в проекте.
  • Включены все необходимые плагины.
  • Используется Unreal Engine версии 5.6 или выше.
  • Декомпилер обновлён до последней версии. Если не работает — попробуйте DEV версию.
