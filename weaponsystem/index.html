Adenex / Weapon System
Weapon System
Unreal Engine Blueprint v1.6
Полностью процедурная система оружия для кинематографических рендеров и игр. Автоматически определяет кости скелета, спавнит вспышки, гильзы, декали, импакты и звуки. Поддерживает запечку через встроенный виджет. A fully procedural weapon effects system for cinematic renders and games. Automatically detects bones, spawns muzzle flashes, shells, decals, impacts, and sounds. Supports baking animations into Level Sequences via a built-in widget.

Setup
  • Init System — Тестовая кнопка. Производит один тестовый выстрел, вызывая все системы спавна (вспышка, гильза, импакт, звук, декали). В финальном сетапе не используется — выстрелы запускаются через события в Sequencer.
  • Reset Bones — Повторно сканирует и определяет нужные кости/сокеты на скелете оружия. Перебирает все имена костей на меше. Используется если оружие было заменено или изменено.
  • Reset Spray — Сбрасывает счётчик выпущенных пуль. Сбрасывает прогрессию паттерна спрея к начальному состоянию.
  • Reset System — Очищает всю карту от декалей, созданных блюпринтом.
  • Is Active — Мастер-переключатель. Если выключен — вся логика стрельбы и эффектов отключается.
  • Is 3rd Person Weapon — Режим оружия от третьего лица. Если включён, смещения и сокеты берутся напрямую с актора оружия. Если выключен — используется POV камера.
  • Is Dual Berettas Mode — Дуальный режим (два оружия в двух руках). Вспышки и гильзы чередуются между двумя костями.
  • POV Camera — Камера для трассировки от первого лица.
  • Weapon — Актор оружия (SkeletalMeshActor). Источник сокетов и эффектов.
  • Recoil Spread — Дополнительный разброс от отдачи. Чем выше значение — тем сильнее разброс.
  • Is Spray — Включает логику паттерна спрея/разброса.
  • Spray Offset (активен при Is Spray) — Текущая интенсивность разброса пуль: смещает конечную точку трассировки. Чем выше значение — тем сильнее отклонение.
  • Is Burst Fire — Режим очереди. Полезен для дробовиков и оружия с несколькими выстрелами за нажатие.
  • Burst Fire Count (активен при Is Burst Fire) — Количество выстрелов в очереди.
  • Enable Wallbangs — Включить пробитие стен.
  • Bullet Energy (активен при Enable Wallbangs) — Начальная энергия пули. Определяет, сколько преград может пробить пуля.
  • Bake Mode — Переключает логику спавна эффектов между runtime-режимом и режимом запечки. Когда включён — декали, импакты и звуки идут по пути запечки в Level Sequence, а не создаются динамически. Не включать вручную — управляется через EUW.
Tracing
  • Trace Length — Дистанция трассировки (диапазон), используется для вычисления конечной точки полёта пули.
  • Debug — Включает визуализацию трассировки.
  • Debug Wall Penetration — Включает визуализацию трассировки специально для логики пробития стен.
  • Actors To Ignore — Акторы, которые трассировка игнорирует (сам актор оружия, персонаж и т.д.).
  • Muzzle Bone Additional Location — Дополнительное смещение позиции кости дула по X/Y/Z. Используется для тонкой подстройки точки вылета пули.
  • Muzzle Bone Additional Rotation — Дополнительный поворот кости дула. Позволяет скорректировать направление выстрела.
  • Tracing Additional Offset — Дополнительное смещение начальной точки трассировки.
Bullets
  • Bullets Enable — Включает спавн физических пуль и трассеров. Пули и трассера видны только в режиме симуляции/плея/рендера.
  • Tracers Every N Shot — Показывает трассер при каждом N выстреле. Для пистолетов и снайперских винтовок обычно используется значение 1, для автоматического оружия обычно используется 3.
  • Bullet Mesh — Меш пули (StaticMesh).
  • Bullet Scale — Масштаб пули.
Muzzle Flash
  • Muzzle Flash Enable — Включает спавн эффектов вспышки на дуле оружия при выстреле.
  • Muzzle Flash Bone — Кость (или сокет), из которой спавнится вспышка (основная).
  • Muzzle Flash Bone 2 — Необязательный второй сокет для дуального режима (например 2 оружия в 2 руках). Если не задан — вспышка спавнится только из основной кости. Можно оставить пустым.
  • Muzzle Flash — Ассет партиклов.
  • Muzzle Flash Rotation — Дополнительный поворот партикла.
  • Muzzle Flash Offset — Дополнительное смещение позиции партикла.
  • Muzzle Flash Scale — Масштаб партикла.
Shells
  • Shells Enable — Включает спавн гильз. Гильзы видны только в режиме симуляции/плея/рендера.
  • Shell Bone — Имя кости, из которой выбрасывается гильза (основной).
  • Shell Bone 2 — Необязательная вторая кость для дуального режима. Если не задана — выброс идёт только из основной кости. Можно оставить пустым.
  • Shell Speed Min / Shell Speed Max — Диапазон начальной скорости гильзы. Реальная скорость выбирается случайно в этом диапазоне.
  • Shells Offset — Смещение точки спавна гильзы относительно кости.
  • Shell Mesh — Меш гильзы (StaticMesh).
  • Shells Rotation — Дополнительное смещение поворота меша гильзы.
  • Shells Mass — Масса гильзы в граммах. Влияет на физику падения.
  • Shells Scale — Масштаб гильзы.
Impact
  • Impacts Enable — Включает спавн партиклов в точке, куда попала пуля. Система автоматически определяет тип поверхности по физическому материалу и выбирает соответствующие партиклы: искры на металле, облако пыли на бетоне, брызги крови на персонаже и т.д. При выключенном параметре никаких визуальных эффектов попадания не спавнится.
  • Default Impact — Система партиклов, когда для поверхности не найден соответствующий физический материал.
  • Impacts Scale — Масштаб для всех партиклов импакта (искры, облака пыли и т.д.).
Time Dilation
  • Muzzle Flash Time Dilation — Замедление/ускорение эффектов вспышек из дула оружия.
  • Impact Time Dilation — Замедление/ускорение эффектов импакта.
  • Sound Time Dilation (Pitch) — Изменяет питч звуков в зависимости от time dilation.
  • TR Time Dilation Multiplier — Глобальный time dilation из ассета TimeRemapping. Передаётся снаружи (например, из Sequencer) и влияет на скорость воспроизведения всех эффектов системы.
Decals
  • Decals Enable — Включить спавн декалей на поверхностях.
  • Decal — Материал декали для обычных поверхностей, когда для у них не найден соответствующий физический материал.
  • Blood Decals Enable — Включает спавн декалей крови на стенах позади персонажей при попадании.
  • Blood Max Distance — Максимальное расстояние между телом и стеной, при котором допускается спавн декаль крови.
Sound
  • Weapon Sound Enable — Включить звук выстрела.
  • Impact Sound Enable — Включить звуки попадания (привязаны к физическому материалу поверхности).
  • Shells Sound Enable — Включить звуки гильз.
  • Weapon Sounds — Звуки выстрела. Из списка выбирается случайный.
  • Volume — Настройки громкости для каждого типа звука.
Setup
  • Init System — Test button. Triggers a single test shot, activating all spawn systems (muzzle flash, shell, impact, sound, decals). Not used in final setup — shots are driven by Sequencer events.
  • Reset Bones — Re-scans and detects the required bones/sockets on the weapon skeleton. Iterates through all bone names on the mesh. Use if the weapon has been replaced or modified.
  • Reset Spray — Resets the fired bullet counter and returns the spray pattern progression to its initial state.
  • Reset System — Clears all decals from the map.
  • Bake Mode — Switches effect spawning logic between runtime mode and baking mode. When enabled, decals, impacts, and sounds are routed into a Level Sequence instead of being spawned dynamically. Do not enable manually — controlled via EUW.
  • Is Active — Master toggle. When false, the system skips all firing and FX logic.
  • Is 3rd Person Weapon — Third-person weapon mode. If enabled, offsets and sockets are taken directly from the weapon actor. If disabled, POV camera is used for tracing.
  • Is Dual Berettas Mode — Dual weapon mode. Muzzle flashes and shells alternate between two bones.
  • POV Camera — Camera used for first-person tracing.
  • Weapon — Weapon actor (SkeletalMeshActor). Source of sockets and effects.
  • Recoil Spread — Additional spread from recoil. Higher values increase dispersion.
  • Is Spray — Enables spray pattern/spread logic.
  • Spray Offset (active when Is Spray is enabled) — Current spray intensity. Offsets the trace endpoint; higher values increase deviation.
  • Is Burst Fire — Burst fire mode. Useful for shotguns and weapons that fire multiple shots per trigger pull.
  • Burst Fire Count (active when Is Burst Fire is enabled) — Number of shots per burst.
  • Enable Wallbangs — Enables wall penetration.
  • Bullet Energy (active when Enable Wallbangs is enabled) — Initial bullet energy. Determines how many surfaces a bullet can penetrate.
Tracing
  • Trace Length — Trace distance (range), used to calculate the ray endpoint.
  • Debug — Enables debug visualization of traces.
  • Debug Wall Penetration — Enables debug tracing specifically for wall penetration logic.
  • Actors To Ignore — Actors ignored by tracing (weapon actor, character, etc.).
  • Muzzle Bone Additional Location — Additional offset of the muzzle bone position (X/Y/Z). Used for fine-tuning the bullet spawn point.
  • Muzzle Bone Additional Rotation — Additional rotation offset for the muzzle bone. Allows adjustment of shot direction.
  • Tracing Additional Offset — Additional offset for the trace starting point.
Bullets
  • Bullets Enable — Enables spawning of physical bullets and tracers. Bullets are visible only in simulation/play/render mode.
  • Tracers Every N Shot — Spawns a tracer on every N shot. For pistols and sniper rifles, default value is usually 1; for automatic weapons 3.
  • Bullet Mesh — Bullet mesh (StaticMesh).
  • Bullet Scale — Bullet scale.
Muzzle Flash
  • Muzzle Flash Enable — Enables muzzle flash effects on firing.
  • Muzzle Flash Bone — Bone (or socket) used as the primary spawn point for the muzzle flash.
  • Muzzle Flash Bone 2 — Optional secondary socket for dual mode. If not set, only the primary bone is used. Can be left empty.
  • Muzzle Flash — Muzzle flash particle system.
  • Muzzle Flash Rotation — Additional rotation offset for the flash.
  • Muzzle Flash Offset — Additional positional offset for the flash.
  • Muzzle Flash Scale — Muzzle flash scale.
Shells
  • Shells Enable — Enables shell ejection. Shells are visible only in simulation/play/render mode.
  • Shell Bone — Name of the bone used as the primary shell ejection point.
  • Shell Bone 2 — Optional secondary bone for dual mode. If not set, only the primary bone is used. Can be left empty.
  • Shell Speed Min / Shell Speed Max — Range of initial shell velocity. The actual value is randomized within this range.
  • Shells Offset — Offset of the shell spawn point relative to the bone.
  • Shell Mesh — Shell mesh (StaticMesh).
  • Shells Rotation — Additional rotation offset for the shell mesh.
  • Shells Mass — Shell mass in grams. Affects physics simulation.
  • Shells Scale — Shell scale.
Impact
  • Impacts Enable — Enables impact particle effects at the bullet hit location. The system automatically detects the surface type via physical material and selects appropriate particles (sparks on metal, dust on concrete, blood on characters, etc.). If disabled, no visual impact effects are spawned.
  • Default Impact — Particle system used when no matching physical material is found.
  • Impacts Scale — Global scale for all impact particles (sparks, dust clouds, etc.).
Time Dilation
  • Muzzle Flash Time Dilation — Slows down or speeds up muzzle flash effects.
  • Impact Time Dilation — Slows down or speeds up impact effects.
  • Sound Time Dilation (Pitch) — Adjusts sound pitch based on time dilation.
  • TR Time Dilation Multiplier — Global time dilation from the TimeRemapping asset. Passed externally (e.g., from Sequencer) and affects playback speed of all system effects.
Decals
  • Decals Enable — Enables decal spawning on surfaces.
  • Decal — Decal material for standard surfaces when no specific physical material is found.
  • Blood Decals Enable — Enables spawning of blood decals on walls behind characters when hit.
  • Blood Max Distance — Maximum distance between the character and a wall for blood decals to spawn.
Sound
  • Weapon Sound Enable — Enables firing sound.
  • Impact Sound Enable — Enables impact sounds (based on surface physical material).
  • Shells Sound Enable — Enables shell casing sounds.
  • Weapon Sounds — List of firing sounds. One is selected randomly.
  • Volume — Volume settings for each sound type.
В папке с блюпринтом WeaponSystem также находится Editor Utility Widget. Он записывает все эффекты системы в Level Sequence покадрово. Делать это необязательно — система работает и без запечки. Но если нужно чтобы результат рендера был одинаковым при каждом запуске, без неё не обойтись: WeaponSystem рандомизирует эффекты при каждом воспроизведении.
  • Weapon Systems — Список акторов WeaponSystem, которые нужно запечь.
Кнопки
  • Analyze — Анализирует все кадры секвенсера и собирает данные о партиклах, декалях и звуках. Показывает статистику: количество партиклов, декалей и звуков по кадрам.
  • Bake — Запекает собранные данные в Level Sequence.
  • Load Active Sequence — Загружает данные из открытого секвенсера и ищет там все WeaponSystem акторы.
  • Reset — Сбрасывает данные виджета в начальное состояние.
The WeaponSystem Blueprint folder also includes an Editor Utility Widget. It records all system effects into a Level Sequence frame by frame. This step is optional — the system works without baking. However, if you need consistent render results across multiple runs, baking is required: WeaponSystem randomizes effects on each playback.
  • Weapon Systems — List of WeaponSystem actors to be baked.
Buttons
  • Analyze — Scans all frames in the sequence and collects data on particles, decals, and sounds. Displays statistics such as the number of particles, decals, and sounds per frame.
  • Bake — Bakes the collected data into a Level Sequence.
  • Load Active Sequence — Loads data from the currently open Sequencer and searches for all WeaponSystem actors.
  • Reset — Resets the widget data to its initial state.
