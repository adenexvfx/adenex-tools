PyToolkit
Unreal Engine Plugin Плагин Unreal Engine v2.1
A plugin for Unreal Engine that adds a robust set of tools for task automation, material workflows, and parameter management via Blueprint and Python.

This plugin is a fork of the Unreal-PyToolkit project (FXTD-ODYSSEY) and has been actively developed as an independent branch since 2021. Required for most widgets to function properly.
Плагин для Unreal Engine, который добавляет удобные инструменты для автоматизации задач, работы с материалами и управления параметрами через Blueprint и Python.

Плагин является форком проекта Unreal-PyToolkit (FXTD-ODYSSEY) и активно развивается как самостоятельная ветка с 2021 года. Необходим для работы большинства виджетов.

  1. Download the plugin version that matches your setup and extract the archive.
    Make sure the plugin version matches your Unreal Engine version (a plugin built for UE 5.7 will not work with UE 5.6).
  2. Close Unreal Engine if it is currently running.
  3. Open your Unreal Engine installation folder and navigate to: ...\UE_FOLDER\Engine\Plugins
  4. Copy the PyToolkit folder into the Plugins directory.
  5. Launch Unreal Engine and open your target project.
  6. Go to Edit → Plugins, search for "PyToolkit," and enable the plugin.
  7. Restart the project to apply the changes.
  1. Скачайте плагин подходящей версии и распакуйте архив.
    Версия плагина должна соответствовать версии Unreal Engine (например, для UE 5.6 не подходит плагин, собранный под 5.7).
  2. Закройте Unreal Engine, если он в данный момент запущен.
  3. Откройте папку установки Unreal Engine и перейдите по пути: ...\UE_FOLDER\Engine\Plugins
  4. Скопируйте папку PyToolkit в каталог Plugins.
  5. Запустите Unreal Engine и откройте нужный проект.
  6. Перейдите в меню Edit → Plugins, введите в строке поиска PyToolkit и включите плагин.
  7. Перезапустите проект, чтобы изменения вступили в силу.
1. PyToolkitBPLibrary — core library
  • PyToolkit|Lib — general editor utilities
    • GetAllObjects — iterates over all loaded UObject instances in editor memory
    • GetAllProperties — returns a list of editable class properties via reflection
    • GetSelectedFolders / SetSelectedFolders — get and set folder selection in the Content Browser
    • SetSelectedAssets — programmatically select assets in the Content Browser
    • CloseEditorForAssets / GetAssetsOpenedInEditor — manage open asset editors
    • SetFolderColor — set folder color in the Content Browser
    • SetViewportLocationAndRotation / GetActiveViewportIndex — viewport control utilities
  • PyToolkit|AssetRegistry
    • GetAssetDependencies(Asset, DependencyType, bRecursive) — retrieve asset dependencies (Hard / Soft / Manage)
    • GetAssetReferencers(Asset, bRecursive) — find which assets reference a given asset
    • FixupRedirectors(FolderPaths) — fix redirectors in specified folders
  • PyToolkit|SequencerAPI
    • GetSequencerSequence — get the Level Sequence currently opened in Sequencer
    • GetSequencerSelectedID — GUIDs of selected objects in Sequencer
    • AddParticleTriggerKey / AddParticleKey / AddParticleActivateKey / AddParticleDeactivateKey — add keys to particle sections with sub-frame precision
    • DisplayFrameToTick / DisplayFloatToTick / TickToDisplayFrame — convert time between DisplayRate and TickResolution
    • GetActorByBindingID — find an actor by Binding GUID
    • SetAnimationSectionWeight — set animation section weight
    • GetDirectorBlueprint / ForceRecompileDirectorBlueprint — manage Director Blueprint
    • CreateTriggerEvent / CreateRepeaterEvent — create Event Section events bound to actor functions
    • MakeLocalBindingID — create a LOCAL binding ID
  • PyToolkit|Socket
    • AddSkeletalMeshSocket — add a socket to a skeletal bone
    • DeleteSkeletalMeshSocket — remove sockets from a skeleton
  • PyToolkit|Skeleton
    • GetSkeletonBoneNum / GetSkeletonBoneName / GetSkeletonBoneIndex — basic bone queries
    • GetSkeletonBoneChildren / GetSkeletonBoneParent — bone hierarchy traversal
    • GetBoneChainToRoot — get bone chain from a given bone to the root
    • GetBoneRefPoseTransform — get ref-pose transform (local or world space)
    • BoneExistInSkeletalMesh / GetAllSkeletalMeshBones / GetAllSkeletonBones
  • PyToolkit|StaticMesh
    • GetStaticMeshLODCount / GetLODTriangleCount — LOD information
    • SetMaterialSlotName — rename a material slot by index
    • RecomputeStaticMeshBounds — recompute bounds after programmatic changes
    • ContentPathToFilesystemPath — convert /Game/... paths to absolute disk paths
  • PyToolkit|WorldLevel
    • GetActorsOfClassInAllSublevels — get all actors of a class across loaded sublevels
    • MoveActorToLevel — move an actor to another level with undo support
    • GetAllLoadedSublevels — list all loaded ULevelStreaming instances
    • SetSublevelVisibilityInEditor — toggle sublevel visibility without triggering streaming
  • PyToolkit|Blueprint
    • GetAllBlueprintFunctions / HasBlueprintCompileErrors / CompileBlueprints
    • AddBlueprintVariable — add a variable with category, tooltip, and Advanced flag in one call
    • FindUnusedBlueprintVariables — find variables not referenced in graphs
    • GetBlueprintVariableTooltip / SetBlueprintVariableTooltip
2. VarByName — property access by name
  • Set/Get by name: float, double, int32, bool, FVector, FLinearColor, UTexture, UTexture2D, UTexture2DArray, UObject, FString, FName
  • GetPropertyTypeByName — return the C++ property type as a string
  • GetPropertyName — wildcard node: plug in a variable directly to retrieve its FName at runtime
  • ResetDirtyProperties — reset only modified properties to CDO defaults
3. TimingBPLibrary — section profiler
  • StartTimedSection — start timing a code section
  • FinishTimedSection — stop timing and return elapsed time
  • LogTimedStats — output a summary report to the log
  • ResetTimedStats — clear all collected stats
