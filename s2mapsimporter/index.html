S2 Maps Importer
Unreal Engine Plugin Unreal Engine Plugin v2.21.2
Native C++ Unreal Engine plugin for importing CS2 maps directly from decompiled .vmdl_c files. Supports static meshes, skeletal meshes, animations, material import, and full map placement via a built-in Slate UI.
Нативный C++ плагин для Unreal Engine, импортирующий карты CS2 напрямую из декомпилированных .vmdl_c файлов. Поддерживает статические меши, скелетные меши, анимации, импорт материалов и полное размещение карты через встроенный Slate UI.

Importing a Map
  1. Decompile the CS2 map using Source2Viewer CLI. The output will contain .vmdl_c and .vmap_c files.
  2. Open the S2 Maps Importer panel in Unreal Engine.
  3. In the Map tab, select the decompiled map folder.
  4. Click Import. The plugin will parse all props, create materials, and place everything in the level.
Importing a Single Model
  1. In the Content Browser, right-click and select Import.
  2. Select a .vmdl_c file.
  3. The plugin will create a StaticMesh or SkeletalMesh asset automatically.
Импорт карты
  1. Декомпилируйте карту CS2 с помощью Source2Viewer CLI. Результат содержит файлы .vmdl_c и .vmap_c.
  2. Откройте панель S2 Maps Importer в Unreal Engine.
  3. На вкладке Map выберите папку с декомпилированной картой.
  4. Нажмите Import. Плагин разберёт все пропсы, создаст материалы и расставит их на уровне.
Импорт одной модели
  1. В Content Browser нажмите правой кнопкой мыши и выберите Import.
  2. Выберите файл .vmdl_c.
  3. Плагин автоматически создаст ассет StaticMesh или SkeletalMesh.
