Adenex / S2 Maps Importer
S2 Maps Importer
Unreal Engine Plugin v2.21.2
Нативный C++ плагин для Unreal Engine, импортирующий карты CS2 напрямую из декомпилированных .vmdl_c файлов.

Скачайте и распакуйте плагин PyToolkit (минимум 1.8 версии) в папку Plugins в корневой директории Unreal Engine (C:Program FilesEpic GamesUE_5.6EnginePlugins). Запустите Unreal Engine и включите следующие плагины (в меню Edit > Plugins):
  • Editor Scripting Utilities
  • Python Editor Script Plugin
  • Sequencer Scripting
  • Http Blueprint
  • PyToolkit
Затем перезапустите движок.
Распакуйте архив с виджетом. Перейдите в папку Content вашего Unreal-проекта (например, D:YourUE5ProjectContent). Скопируйте все подпапки из архива в папку Content проекта.
Удалите старую версию аддона SourceIO из %appdata%Blender FoundationBlender4.0scriptsaddons. Откройте Blender и установите кастомную версию SourceIO из архива S2 Maps Tool.
Запустите Unreal Engine. В Content Browser найдите папку Widgets. Кликните по виджету S2 Maps Tool правой кнопкой мыши и выберите Run Editor Utility Widget.
Если возникают ошибки, проверьте следующее:
  • Виджет установлен по правильному пути внутри проекта Unreal Engine;
  • Все необходимые плагины включены в настройках проекта;
  • Используется Unreal Engine версии 5.6 или новее;
  • Декомпилер обновлён до последней версии.
    Если стандартная версия не работает, рекомендуется попробовать DEV-версию;
  • Используется Blender версии 4.0 для конвертации мешей.
  • Game — выбор игры, файлы которой будут использоваться при импорте.
  • Blender Executable — путь к исполняемому файлу blender.exe.
  • Decompiler Executable — путь к исполняемому файлу декомпилера. Скачайте cli-windows-x64.zip, распакуйте и укажите путь к .exe.
  • Game Folder — корневая папка установленной игры (например D:SteamLibrarysteamappscommonCounter-Strike Global Offensive).
  • Game Decompiled Folder — папка, куда будут сохраняться декомпилированные файлы карты. Можно выбрать любое удобное место.
  • Extended Log — включить расширенный вывод логов (нужно при отладке проблем).
  • Buttons Color — цвет кнопок интерфейса виджета.
  • Map VPK File — путь к VPK-файлу карты (например, .../csgo/maps/de_dust2.vpk). Нажмите ... чтобы выбрать файл.
  • Map Name — внутреннее имя карты. Обычно подставляется автоматически из названия уровня внутри VPK-файла.
  • Map Scale — масштаб импортируемой карты.
  • Spawn Cables — создавать ли кабели/провода на карте.
  • Spawn Environment Light — создавать ли источник окружающего освещения.
  • Spawn Light Entities — создавать ли точечные источники света.
  • Save The Level During Import — автоматически сохранять уровень после импорта.
  • Max Texture Size — максимальное разрешение текстур при импорте.
Процесс импорта карты:
  1. Выберите VPK-файл через кнопку ... рядом с Map VPK File
  2. Проверьте Map Name — не должно быть пустым полем
  3. Нажмите Decompile map — декомпилирует геометрию и ассеты
  4. Нажмите Convert map and props — конвертирует модели с помощью Blender в формат FBX
  5. Нажмите Import map — импортирует карту в проект
Если источники света перекрываются геометрией или расположены неправильно: перейдите во вкладку Misc → Light и нажмите Fix stuck light.
В движке Source2 большинство мешей сгруппированы вместе для оптимизации. Например, этот ящик на de_vertigo. Было сгруппировано в один меш около 6 ящиков. Чтобы подвинуть только один объект, необходимо включить режим Modelling, перейти на вкладку Model → PolyGroup Edit. Далее выбрать режим вертексов, выделить все точки объекта и подвинуть его с помощью пивота. После редактирования меша, необходимо нажать Accept по центру экрана, чтобы сохранить его позицию.
