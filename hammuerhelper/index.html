Hammuer Helper
Widget Виджет v1.1
Widget for preparing Source 1 maps for import into Unreal Engine using the HammuEr plugin. Cleans up model files, optionally distributes them across folders, and retargets materials.
Виджет для подготовки Source 1 карт к импорту в Unreal Engine с использованием плагина HammuEr. Очищает файлы моделей, опционально распределяет их по папкам и ретаргетит материалы.

Clean Tab — File Cleanup
  1. In the Models Folder field, specify the folder with map models (use the ... button).
  2. Enable Split Models if you want to distribute models across multiple folders for organized import.
  3. If Split Models is enabled, set Folders Count — the number of folders to split into.
  4. Click Run.
After this, all unnecessary files will be removed, leaving only what is needed for import: .mdl, .dx90.vtx, .phy, .vvd.
Retarget Tab — Material Replacement
  1. In the Old Material field, select the standard HammuEr material.
  2. In the New Material field, select the material to switch all material instances to.
  3. Click Retarget.
The original HammuEr material is preserved — only the assignment is changed.
Вкладка Clean — очистка моделей
  1. В поле Models Folder укажите папку с моделями карты (кнопка ...).
  2. Если хотите разложить модели по папкам — включите Split Models.
  3. При включённом Split Models укажите Folders Count — на сколько папок разбить.
  4. Нажмите Run.
После этого из папки удалятся все лишние файлы, останется только нужное для импорта: .mdl, .dx90.vtx, .phy, .vvd.
Вкладка Retarget — замена материала
  1. В поле Old Material выберите стандартный материал HammuEr.
  2. В поле New Material выберите новый материал, на который нужно переключить все материал инстансы.
  3. Нажмите Retarget.
Оригинальный материал HammuEr сохраняется — меняется только назначение.
